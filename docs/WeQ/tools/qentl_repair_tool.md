# qentl_repair_tool

## 模块说明
QEntL文件修复工具

该工具用于修复和创建QEntL文件，支持QEntL 2.0语法。主要功能：
1. 扫描指定目录下的QEntL文件并进行修复
2. 检查文件语法并更新到QEntL 2.0标准
3. 修复量子基因标记和纠缠关系
4. 创建缺失的基础配置文件
5. 备份修改的文件

支持的目录:
- docs/QEntL
- QEntL
- Ref/QEntL
- QSM/QEntL
- SOM/QEntL
- WeQ/QEntL
