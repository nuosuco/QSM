# 防欺骗+防休眠系统机制

## 核心规则
1. **诚实汇报**：做了什么就说什么，没做就说没做
2. **持续工作**：每10分钟必须有实际工作产出
3. **详细记录**：每次汇报必须说明具体做了什么

## 工作机制

### 1. 定时汇报（每10分钟）
- 检查git提交记录
- 列出具体完成的工作
- 说明当前进展

### 2. 并行工作
- 助手负责核心模块开发
- 子代理A负责QNS训练
- 子代理B负责QDFS扩展
- 子代理C负责QVM扩展
- 子代理D负责四大模型

### 3. 监督机制
- 每10分钟自动检查项目状态
- 发现长时间无活动立即报警
- 发现欺骗行为立即纠正
