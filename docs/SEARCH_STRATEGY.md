# SOM 松麦 - 商品搜索策略方案

## 一、核心理念

中医小麦SOM通过全方位的辨证诊断，从饮食、药膳、生活起居、健身锻炼等方面全面推荐。平台只推荐有机认证、环保认证的最健康产品，未来目标是线上线下同步开设超市，品类一致。

## 二、搜索关键词体系

### 2.1 前缀机制（_ensure_organic）

所有搜索关键词自动追加前缀 `有机`，确保搜索结果优先展示有机认证产品。

```
前缀：有机
示例：点"谷物杂粮" → 实际搜索"有机 五谷杂粮 大米 小米 燕麦 藜麦 原生态 野生"
```

**为什么前缀不加"原生态 野生"？**
原生态和野生没有国家认证标准，作为前缀会导致搜索结果不精准。但作为分类关键词放在后面，可以辅助匹配到天然好产品。

### 2.2 分类关键词结构

每个分类关键词由三部分组成：
1. **品类词**：如"五谷杂粮 大米 小米 燕麦 藜麦"
2. **品质词**：如"原生态 野生"
3. **前缀**：自动加"有机"

### 2.3 多关键词拆分搜索策略（_split_keywords）

**为什么需要拆分？**

淘宝/京东的搜索API对关键词长度有限制，一次性传太多词（超过3-4个）会搜不到结果或返回空。但我们的分类关键词可能包含10-20个词（如"全部"有14个词），所以必须拆成短组合分别搜索再合并。

**拆分规则：**

```
输入：有机 食品 养生 食材 滋补品 药膳 原生态 野生 日用品 家居 绿植 健身 保健 棉品 美妆

拆分步骤：
第一步：取前2个词作为基础组 → "有机 食品"
第二步：遍历剩下的每个词，依次和第一个词组合 → "有机 养生"、"有机 食材"、"有机 滋补品"...

最终拆分为以下短组合分别搜索：
  1. 有机 食品          ← 基础组（保底一定有结果）
  2. 有机 养生          ← 覆盖养生品类
  3. 有机 食材          ← 覆盖食材品类
  4. 有机 滋补品        ← 覆盖滋补品
  5. 有机 药膳          ← 覆盖药膳
  6. 有机 原生态        ← 覆盖原生态
  7. 有机 野生          ← 覆盖野生
  8. 有机 日用品        ← 覆盖日用品
  9. 有机 家居          ← 覆盖家居
  10. 有机 绿植         ← 覆盖绿植
  11. 有机 健身         ← 覆盖健身
  12. 有机 保健         ← 覆盖保健
  13. 有机 棉品         ← 覆盖棉品
  14. 有机 美妆         ← 覆盖美妆

去重规则：
- 用 item_id 去重，同一商品不会重复出现
- 合并后最多取 page_size * 2 个商品（防止太多）
- 每个子搜索的 page_size = 原始 page_size
```

**代码实现：**

```python
def _split_keywords(self, keyword: str) -> list:
    """将长关键词拆分成2-3个词的短组合，确保淘宝能搜到"""
    words = keyword.split()
    if len(words) <= 3:
        return [keyword]  # 3个词以内不用拆
    # 取前2个词作为基础，然后依次追加
    result = []
    base = ' '.join(words[:2])
    result.append(base)
    for i in range(2, len(words)):
        result.append(f"{words[0]} {words[i]}")
    return result
```

**注意：** 为什么基础组用前2个词而不是第1个？因为前缀"有机"是第1个词，品类词是第2个词，这样"有机 食品"作为基础组是最通用的搜索，保底能搜到东西。

## 四、排除过滤机制

### 4.1 排除关键词列表（EXCLUDE_KEYWORDS）

搜索结果中出现以下关键词的商品将被过滤掉，排除顺序按优先级：

```
第一优先级 - 书籍文具：书、书籍、教材、课本、图书、文具、笔记本、笔、本子
第二优先级 - 化工化学：化工、化学、试剂、肥料、农药、化肥、工业、原料
第三优先级 - 玩具娱乐：玩具、模型、手办、乐高、积木、游戏、桌游、卡牌
第四优先级 - 电子产品：手机、电脑、平板、耳机、充电器、数据线、电子、数码、电器
第五优先级 - 汽车配件：汽车、轮胎、机油、车、配件、改装
第六优先级 - 宠物用品：猫粮、狗粮、宠物、猫砂
第七优先级 - 其他：塑料、包装、纸箱、胶带
```

### 4.2 过滤逻辑

```python
def _is_excluded(self, item: dict) -> bool:
    """检查商品标题是否包含排除词"""
    title = item.get('title', '').lower()
    for ex in EXCLUDE_KEYWORDS:
        if ex in title:
            return True
    return False
```

## 五、搜索流程（完整链路）

```
用户点击分类 → 获取分类关键词
         ↓
    _ensure_organic() 自动加"有机"前缀
         ↓
    _split_keywords() 拆成2词短组合
         ↓
    分别调用淘宝API搜索 → 合并结果
         ↓
    _is_excluded() 过滤无关商品
         ↓
    返回前端展示
```

## 六、注意事项

1. **分类关键词不含"有机"**：前缀自动加，避免重复
2. **关键词长度控制**：淘宝单次搜索不超过3个词，过长用拆分策略
3. **排除词匹配标题**：只匹配商品标题，不匹配描述
4. **分类名 vs 关键词**：分类名显示给用户，关键词用于后端搜索，两者可以不同
5. **未来扩展**：可以接入更多平台（拼多多、抖音电商等），保持过滤逻辑一致
