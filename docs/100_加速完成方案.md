### 1.2 关键瓶颈

**核心瓶颈：QVM缺少运行时函数支持**

QCL引导器(QCL_main.qentl)已被QVM成功启动并注册15个函数，但：
- 调用 `print()` → QVM不认识 → 失败
- 调用 `len()` → QVM不认识 → 失败
- 调用 `get_directory_entries()` → QVM不认识 → 失败
- 调用 `read_source_file()` → QVM不认识 → 失败

**解决方案：在QVM的OP_FUNC_CALL_STMT中添加运行时函数路由**
```c
// 在qvm_bootstrap.c的OP_FUNC_CALL_STMT处理器中：
if (strcmp(func_name, "print") == 0) {
    // 从栈上取参数，执行打印
} else if (strcmp(func_name, "len") == 0) {
    // 返回数组长度
} else if (strcmp(func_name, "get_directory_entries") == 0) {
    // 扫描目录
} else if (strcmp(func_name, "read_source_file") == 0) {
    // 读取文件
}
```
