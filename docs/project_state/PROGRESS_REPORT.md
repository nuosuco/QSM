## 项目概述
QSM（量子叠加态模型）是一个完全自主研发的量子计算生态系统，包含：
- QEntL量子编程语言
- QVM量子虚拟机
- QNN量子神经网络
- 彝文训练数据管道
- 量子动态文件系统
