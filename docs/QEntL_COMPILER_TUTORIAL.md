# QEntL编译器使用教程

## 快速开始

### 第一步：编写源代码

创建一个 `.qentl` 文件，例如 `hello.qentl`:

```
配置 {
    版本: "1.0.0"
    作者: "中华"
}

函数 主函数() {
    let 消息 = "你好，量子世界！"
    返回 消息
}
```

### 第二步：编译

```bash
python3 /root/QSM/tools/qentl_compile.py hello.qentl
```

### 第三步：运行

```bash
python3 /root/QSM/tools/qbc_vm.py hello.qbc
```

## 基础语法

### 1. 配置块

```
配置 {
    版本: "1.0.0",
    调试: true
}
```

### 2. 类型定义

```
类型 用户 {
    名称: 字符串,
    年龄: 整数,
    活跃: 布尔
}
```

### 3. 函数定义

```
函数 加法(a: 整数, b: 整数) -> 整数 {
    let 结果 = a + b
    返回 结果
}
```

### 4. 控制流

```
函数 判断(x: 整数) {
    如果 x > 0 {
        返回 "正数"
    } 否则 {
        返回 "非正数"
    }
}
```

### 5. 循环

```
函数 累加(n: 整数) {
    let 总和 = 0
    循环 i 在 范围(n) {
        let 总和 = 总和 + i
    }
    返回 总和
}
```

### 6. 量子类

```
quantum_class 量子处理器 {
    函数 初始化() {
        返回 "就绪"
    }
    
    函数 执行(指令: 字符串) {
        返回 指令
    }
}
```

## 示例代码

### 示例1：量子加法器

```
配置 {
    模式: "量子计算"
}

函数 量子加法(a: 整数, b: 整数) {
    let 结果 = a + b
    返回 结果
}
```

### 示例2：条件判断

```
配置 {
    调试: true
}

函数 奇偶判断(n: 整数) {
    如果 n % 2 == 0 {
        返回 "偶数"
    } 否则 {
        返回 "奇数"
    }
}
```

### 示例3：量子类应用

```
quantum_class 量子寄存器 {
    函数 存储数据(数据: 字符串) {
        返回 数据
    }
    
    函数 读取数据() {
        返回 "存储的数据"
    }
}
```

## 常见问题

### Q: 如何调试编译错误？

A: 查看错误信息中的行号和列号，定位问题位置。

### Q: 支持哪些数据类型？

A: 目前支持：字符串、整数、布尔、数组、映射。

### Q: 如何使用量子操作？

A: 使用 `quantum_class` 定义量子类，在类中实现量子操作。
