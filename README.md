# QSM量子叠加态模型

## 项目概述

QEntL（Quantum Entanglement Language）是基于量子叠加态模型的编程语言生态系统，包含编译器、虚拟机、运行时环境、模型系统和安装器于一体的完整编程平台。


## 四大量子模型

| 模型 | 名称 | 功能 |
|------|------|------|
| QSM | 量子叠加态模型 | 主模型，协调所有子系统 |
| SOM | 松麦经济模型 | 经济计算与资源分配 |
| WeQ | 量子通讯模型 | 量子纠缠通讯网络 |
| Ref | 量子反省模型 | 自我优化与错误纠正 |
