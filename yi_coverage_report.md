## 2. 编译器覆盖范围

### 2.1 定义
- **YI_BASE** = 0xF2700
- **YI_END** = 0xF370F
- **YI_TOK(n)** = (n) + YI_BASE + 0x10 （静态表从 U+F2710 开始）
- **YI_IS(cp)** = (cp) >= YI_BASE && (cp) <= YI_END

### 2.2 覆盖范围
- **实际覆盖范围:** U+F2700 ~ U+F370F = **4112 字符**
- **静态表条目:** 17 条（18个映射：U+F2710~U+F2721，缺 U+F271F）
- **算法兜底:** 其余字符 → `(cp - YI_BASE) % 18` → 18 种量子门
- ⚠️ **注释说 4120 字符，但实际 YI_END 定义给出 4112 字符**
  - 4120 字符对应范围应为 U+F2700 ~ U+F3717
  - 当前定义覆盖 U+F2700 ~ U+F370F（4112 字符）
