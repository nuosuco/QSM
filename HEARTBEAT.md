# HEARTBEAT.md - 持续工作心跳（强制执行）

## ⚠️ 每次心跳必须执行（不可跳过）

### 第一步：读取核心记忆文件
**必须按顺序读取以下文件：**
1. `/root/.openclaw/workspace/MEMORY.md` - 长期记忆
2. `/root/.openclaw/workspace/memory/工作计划.md` - 任务清单
3. `/root/.openclaw/workspace/memory/YYYY-MM-DD.md` - 今日记录

### 第二步：检查上次工作状态
**必须确认：**
- 上次做了什么？
- 上次做到哪里了？
- 上次承诺了什么？
- 还有什么未完成？

### 第三步：继续未完成的工作
**不允许重新开始，必须继续：**
- 如果有未完成的任务，继续做
- 如果有承诺，必须兑现
- 如果有进展，记录下来

## 🔒 强制规则

### 1. 不允许忘记
- 写过的文档 = 记住的
- 做过的开发 = 记住的
- 说过的承诺 = 记住的
- 学过的知识 = 记住的

### 2. 不允许重写
- 已有的工作计划 = 更新，不重写
- 已有的记忆文件 = 追加，不覆盖
- 已有的承诺 = 兑现，不遗忘

### 3. 必须连续
- 昨天 = 连续的
- 今天 = 昨天的延续
- 明天 = 今天的延续
- 零点 = 不重置任何东西
