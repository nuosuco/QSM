📚 Language Features

Variables & Types

x = 42 s = "你好" arr = [1, 2, 3] d = {"键": "值"}

Control Flow

如果 x > 5 { 打印("大") } 否则 { 打印("小") } 循环 i 在 0 到 10 { 打印(i) } 循环当(x < 10) { x = x + 1 } 匹配 x { 情况 1 { ... } 默认 { ... } }

Functions

函数 加法(a, b) { 返回 a + b } 结果 = 加法(3, 5) // = 8

Functional Programming

映射(x -> x * 2, arr) // map 过滤(x -> x > 3, arr) // filter 归约((a,b) -> a+b, arr) // reduce 排序(arr) // sort 反转(arr) // reverse 去重(arr) // dedupe

String Operations

s = "hello" s * 3 // "hellohellohello" 长度(s) // 5 子串(s, 1, 3) // "el" 分割(s, "l") // ["he","","o"] 替换(s, "l", "L") // "heLLo"

Math
