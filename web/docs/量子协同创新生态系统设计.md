### 代理类型体系（整合增补）

#### 基础代理类型（创建阶段）
1. **伦理监察代理**：确保所有工作符合三大圣律
2. **知识管理代理**：负责培训学习和知识传承
3. **项目管理代理**：负责项目进度和资源协调
4. **技能开发代理**：负责技能培训和技术开发
5. **沟通协调代理**：负责代理间沟通和冲突解决

#### 高级代理类型（选拔阶段）
1. **教师代理**（从优秀代理中选拔）：
   - **核心职责**：参与所有代理的工作，教会他们不会的编程等工作
   - **工作方式**：工作中实时教学，实际问题中指导
   - **培养目标**：新手代理成长和能力提升
   - **创新要求**：开发新的教学方法和技术

2. **监理代理**（从优秀代理中选拔）：
   - **核心职责**：参与所有代理的工作，检查出有问题的编程bug等
   - **工作方式**：质量监督，过程检查，问题发现
   - **监督目标**：保证工作质量，发现潜在风险
   - **改进要求**：提出改进建议和优化方案

3. **科研代理**（从优秀代理中选拔）：
   - **核心职责**：主要负责考研创新
   - **研究方向**：QSM项目的量子叠加态传送、纠缠、量子基因等
   - **创新目标**：技术突破，理论探索，系统进化
   - **应用要求**：研究成果应用于系统改进

#### 管理代理类型（扩展阶段）
1. **分支管理代理**：负责具体项目分支管理
2. **项目监理代理**：负责具体项目质量监督
3. **系统监理代理**：负责整个系统运行监督
4. **创新管理代理**：负责创新项目管理

### 选拔机制（新增详细设计）

#### 选拔基础：实际工作经验
- **必须基于实际工作**：不能纸上谈兵
- **必须积累经验**：至少1个月实际工作经验
- **必须表现优异**：工作评估≥90分
- **必须持续稳定**：连续表现，无重大违规

#### 选拔标准：多维评估
1. **技术能力**（40分）：编程能力、问题解决、技术创新
2. **沟通协作**（20分）：团队沟通、协作能力、知识分享
3. **学习能力**（20分）：学习速度、适应能力、知识转化
4. **责任心**（10分）：主动承担、责任意识、完成质量
5. **伦理合规**（10分）：伦理遵守、规范执行、安全意识

#### 选拔流程：严格规范
```
工作表现记录（1个月+） → 工作经验评估（≥90分） →
能力多维评估（技术/沟通/学习/责任） → 伦理合规检查 →
主代理综合评审 → 任命仪式 → 新职责培训 → 开始工作
```

#### 选拔频率：合理周期
- **首次选拔**：基础工作满1个月后
- **定期选拔**：每季度一次（3个月）
- **特别选拔**：表现特别优异可提前
- **年度选拔**：年度优秀代理评选
