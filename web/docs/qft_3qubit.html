# 量子傅里叶变换 (QFT) - 3-qubit 版本 # QFT 是 Shor 算法的核心组件 # 对 |000> 到 |111> 的叠加态做 QFT init 3 # QFT(qubit 0): H + 受控旋转 H 0 # 受控 R2 (控制=0, 目标=1): 简化为 CNOT (实际应该有相位门) CNOT 0 1 # QFT(qubit 1): H + 受控旋转 H 1 # 受控 R2 (控制=1, 目标=2) CNOT 1 2 # QFT(qubit 2): H H 2 # 交换量子比特顺序 (3-qubit QFT 需要反转) # 简化: 直接测量 MEASURE 0 0 MEASURE 1 1 MEASURE 2 2 PRINT 0 STOP
