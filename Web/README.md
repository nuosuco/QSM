# QEntL Web操作系统

## 概述

这是QEntL量子操作系统的Web版本，用户可以通过浏览器直接使用完整的量子操作系统功能。

## 🌍 三语支持

本系统原生支持三种语言：

| 语言 | 代码 | 说明 |
|------|------|------|
| 🇨🇳 中文 | zh-CN | 东方智慧表达 |
| 🇺🇸 英文 | en-US | 国际技术标准 |
| ⛰️ 滇川黔贵通用彝文 | yi | 古老智慧传承 |

## 📁 目录结构

Web/
├── index.html              # Web操作系统桌面主页
├── apps/                   # 应用程序目录
│   ├── qvm/               # 量子虚拟机应用
│   ├── compiler/          # QEntL编译器
│   ├── files/             # 文件管理器
│   ├── settings/          # 设置中心
│   ├── social/            # 社交网络（WeQ）
│   ├── economy/           # 经济系统（SOM）
│   ├── monitor/           # 系统监控（Ref）
│   └── store/             # 应用商店
├── assets/                 # 资源文件
│   ├── css/               # 样式表
│   ├── js/                # JavaScript脚本
│   ├── fonts/             # 字体文件（含彝文字体）
│   └── icons/             # 图标资源
└── api/                    # 后端API接口

## 🚀 部署方式

开发环境:
- 源码位置: /root/QSM/Web/
- Web服务器: 软链接到 /var/www/som.top

Git管理:
- 与QSM项目一起提交到远程仓库

## 🌟 特色功能

1. 三语原生支持 - 界面、文档、编程均支持三语
2. 量子虚拟机 - 8量子比特模拟器
3. 三语编译器 - 支持中英彝混合编程
4. 量子态指示 - 实时显示系统量子状态
5. 应用生态 - 可扩展的量子应用系统
