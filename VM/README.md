## 快速开始

### 命令行启动
```bash
cd /root/QSM/VM
python3 src/main.py
```

### 可用命令
- `help` - 显示帮助
- `status` - 显示虚拟机状态
- `run` - 运行QBC字节码
- `quit` - 退出
