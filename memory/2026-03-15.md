## 新学习内容

### 1. QEntL编译器架构
```
源代码(.qentl) → 词法分析 → Token流 → 语法分析 → AST → 语义分析 → 优化AST → 字节码(.qbc)
```

### 2. QSM服务核心组件
- **QuantumState接口**: 量子状态接口，定义getId()、getSuperpositionStates()、collapse()等方法
- **ConcreteQuantumState类**: 量子状态具体实现，包含叠加态概率管理
- **QuantumStateManager类**: 量子状态管理器，处理状态创建、删除、纠缠连接
- **QSMService类**: QSM主服务，整合状态管理、纠缠处理、转换引擎

### 3. 量子字节码文件格式
- 魔数与版本信息 (16字节)
- 量子基因编码段 (32字节)
- 指令段 (可变长度)
- 数据段 (可变长度)
- 符号表 (可变长度)
- 量子纠缠信息 (可变长度)

### 4. QEntL运行时API架构
- **内存管理API**: qentl_alloc, qentl_free, qentl_gc_collect
- **类型系统API**: QentlType枚举（NULL/INTEGER/FLOAT/BOOLEAN/STRING/ARRAY/OBJECT/FUNCTION）
- **执行引擎API**: qentl_bytecode_load, qentl_execute, qentl_runtime_init
- **标准库API**: qentl_print, qentl_array_push, qentl_string_concat
- **错误处理API**: qentl_error_set, qentl_throw, qentl_try_catch
- **模块系统API**: qentl_module_load, qentl_export, qentl_import
