# QSM日报 2026-05-02

## 🔥🔥🔥 QEntL重大突破

### 编译器(qentl_compiler_v3.py)
1. **多重否则如果(elif)代码生成修复** — 之前只生成前2个elif分支，第3+个被忽略！
   - 根因: If代码生成只处理children[2]，忽略children[3..n]
   - 修复: 3个子节点→递归生成嵌套If; 4+子节点→逐个生成elif条件+跳转
   - 添加了end_label LABEL指令(之前完全缺失!)
2. **顶层层级扫描修复** — run_with_function不再执行函数体作为顶层代码
   - 根因: 扫描从IP=0到func_ip的所有指令，包括函数体
   - 修复: 只扫描到第一个函数边界(all_func_ips[0])

### 虚拟机(qbc_vm.py)
1. **🔥 CALL/RETURN参数保存恢复** — 递归函数终于正确！
   - 根因: flat namespace下递归函数修改了调用者的参数
   - 例: 斐波那契(n-1)修改n后，斐波那契(n-2)用了错误的n
   - 修复: CALL时保存saved_params={}, RETURN时恢复
   - 新增: self.call_stack_params列表
2. **斐波那契双递归**: fib(0)=0, fib(5)=5, fib(10)=55 ✅
3. **GCD欧几里得**: gcd(12,8)=4, gcd(100,75)=25 ✅
4. **四则运算计算器**: 10+3=13, 10-3=7, 10*3=30, 10/3=3.33 ✅
