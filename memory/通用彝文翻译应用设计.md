# 通用彝文翻译应用 - 设计文档

**版本**: v1.0  
**日期**: 2026-03-09  
**状态**: 开发中

---

## 🎯 功能目标

### 核心功能
1. **三语互译** - 中文 ↔ 英文 ↔ 通用彝文
2. **实时翻译** - 输入即时转换
3. **量子模型集成** - 四个量子模型智能翻译

### 技术架构
- 前端：Web界面（HTML/CSS/JavaScript）
- 后端：Python Flask API
- 数据：4120个通用彝文字符映射表

---

## 📊 数据结构

### 三语映射表
```json
{
  "yi": "U+F2970",  // 彝
  "zh": "彝",
  "en": "Yi"
}
```

### 应用结构
```
/var/www/som.top/apps/translator/
├── index.html      # 主页面
├── api.py          # Python API
├── yi_dict.json    # 彝文字典
└── static/
    └── styles.css  # 样式
```
