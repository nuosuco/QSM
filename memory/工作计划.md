# 工作计划.md - QEntL量子生态

## 项目核心理解

**QSM = 量子叠加态模型** (核心AI)  
**QEntL = 量子操作系统** (基础环境)

## 🎯 正确的量子开发流程

```
.qentl源码 → 量子编译器 → .qbc字节码 → QEntL虚拟机 → 执行训练
```

## QEntL量子生态架构

```
/root/QSM/
├── QEntL/                    # 量子操作系统
│   ├── bin/qentl            # C语言启动器
│   └── System/
│       ├── VM/*.qentl       # 量子虚拟机核心
│       ├── Compiler/*.qentl # 量子编译器核心
│       └── Kernel/filesystem/*.qentl # 量子动态文件系统
├── Models/                   # 量子模型
│   ├── QSM/                 # 量子叠加态模型
│   ├── SOM/                 # 松麦币经济模型
│   ├── WeQ/                 # 对话协调模型
│   ├── Ref/                 # 自反省模型
│   └── training/*.qentl     # 量子训练程序
└── Web/                      # Web前端
    └── data/*.json          # 数据文件
```
