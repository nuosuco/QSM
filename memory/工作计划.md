## 📁 Web操作系统代码位置

### 开发版本（随时开发）
```
/root/QSM/Web/                    # Web操作系统开发版
├── index.html                    # 桌面主页
├── apps/                         # 应用目录
│   ├── qvm/                      # 量子虚拟机应用
│   ├── compiler/                 # 编译器应用
│   ├── files/                    # 文件管理器
│   └── ...
├── assets/                       # 资源文件
└── README.md                     # Web系统说明
```

### 服务版本（用户访问）
```
/var/www/som.top/                 # Web服务版本（用户访问）
```

### 升级流程
```bash
# 开发完成后执行升级脚本
cd /root/QSM
bash scripts/upgrade_web.sh
```

### 升级频率
- **Web版本**: 每天晚上升级一次（覆盖服务版本）
- **虚拟机版本**: 按版本号发布更新
- **操作系统版本**: 按大版本发布更新

---

## 📊 开发与服务分离原则

1. **开发版本** (`/root/QSM/Web/`) - 随时开发，不影响用户
2. **服务版本** (`/var/www/som.top/`) - 稳定运行，用户访问
3. **升级机制** - 开发完成 → 测试通过 → 覆盖部署
