## 🎯 项目本质（永远记住）

**QSM是一个量子操作系统，不是普通项目！**

三种部署方式只是运行环境不同：
- 原生部署 = 直接在硬件上运行
- 虚拟机部署 = 在QVM上跨平台运行

**核心不是"部署"，而是"运行.qentl程序"！**
