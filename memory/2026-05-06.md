2. **修复方案**: 加载全部数据(diff<=5), 每epoch动态过滤
3. **Optimizer BUG**: resume后optimizer被新AdamW覆盖
4. **修复方案**: optimizer创建移到resume之前
5. **Subset BUG**: random_split包装后无法调用set_max_difficulty
6. **修复方案**: 保存full_ds引用=原始SPMDataset

### train_v14_alibi.py 关键改动
- SPMDataset新增 self.raw + set_max_difficulty()方法
- 初始加载max_difficulty=5(全部数据)
- 每epoch调用get_max_difficulty()→动态过滤
