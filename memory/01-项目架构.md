# QSM量子操作系统架构

## 三层架构
```
应用层：QSM量子叠加态模型（用户界面）
    ↓
系统层：QEntL量子操作系统
    ├── 四大量子模型（QSM/SOM/WeQ/Ref）
    ├── 量子内核
    └── 量子编译器
    ↓
基础层：量子虚拟机 + 量子文件系统
```

## 五大量子模型
| 模型 | 功能 | 状态字符 |
|------|------|----------|
| QSM | 量子状态管理 | 心 |
| SOM | 松麦经济 | 凑 |
| WeQ | 量子通信 | 连接 |
| Ref | 系统监控 | 选择 |
| QEntL | 操作系统核心 | 王 |

## 三种部署
1. 原生部署 - 直接在硬件运行
2. 虚拟机部署 - 在QVM跨平台运行
3. Web部署 - 在浏览器运行

核心：都要能运行.qentl文件
