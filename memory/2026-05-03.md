1. **FieldAccess赋值**: q.状态 = 1 → FieldAssign AST节点
   - 解析: expression statement → field access → detect `=` → FieldAssign
   - 代码生成: LOAD obj → LOAD value → STORE_FIELD field_name
2. **FieldAssign节点**: 新AST类型，children=[object, value], value=field_name

## 凌晨追加3 (04:16-04:50)

### 🔥 QEntL异常处理完整实现！

#### 抛出/捕获(throw/catch)
- **TokenType.THROW**: 新增枚举类型 + '抛出'/'throw'关键字
- **_parse_statement**: THROW → Throw AST节点(表达式子节点)
- **代码生成**: gen(exception_value) + THROW opcode
- **Try代码生成修复**: catch_label后添加STORE_VAR绑定异常变量
