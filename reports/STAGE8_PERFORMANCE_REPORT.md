> 执行时间：2026-07-07 | 全量QVM 548/548 PASS | 基线自举链 57ms PASS

## 1. 优化方案

| # | 优化项 | 实现 |
|---|--------|------|
| 1 | **mmap 替代 fopen** | QCL编译器 + QVM执行器均改用 `open()`+`mmap()`+`MADV_SEQUENTIAL`，零拷贝原地解析 |
| 2 | **小buffer** | `MAX_LINE_LEN` 4096→1024；编译器逐行 `memchr` 定位换行符，栈内存降低60% |
| 3 | **静态编译 + strip** | `-O2 -static -lm` + `strip` 生成完全独立二进制（含libc，670KB） |
| 4 | **-Os 动态编译** | `-Os -lm` + `strip` 生成最小组件二进制（保持动态链接，适合容器部署） |
| 5 | **编译优化** | 简化 `if/strncmp` 链、消除冗余变量、`memchr`替代`fgets` |

生成文件：
- `src/qcl_bootstrap_opt.c` — 优化版QCL编译器（mmap + 小buffer）
- `src/qvm_bootstrap_opt.c` — 优化版QVM执行器（mmap + MADV_SEQUENTIAL）
- `tools/generate_stress_test.sh` — 压力测试用例生成器
- `tools/generate_controlflow.sh` — 控制流压力测试生成器

## 2. 二进制大小对比

| 组件 | 基线 (-O2) | -Os+strip | 静态+strip | 备注 |
|------|-----------|-----------|-----------|------|
| qcl_bootstrap | **12,984 B** (12.7KB) | 10,640 B (10.4KB) ↓18% | 678,248 B (662KB) ↑4976% | 动态→静态含libc |
| qvm_bootstrap | **34,376 B** (33.6KB) | 22,960 B (22.4KB) ↓33% | 698,728 B (682KB) ↑1936% | 动态→静态含libc |

**推荐方案：**
- 容器/嵌入式部署 → **-Os+strip 动态版**（最小组件）
- 独立部署/离线环境 → **静态+strip**（零依赖，但体积大）

## 7. 综合结论

| 维度 | 结论 |
|------|------|
| **最佳方案** | -O2 + mmap + MADV_SEQUENTIAL（平衡性能/大小） |
| **最小部署** | -Os + strip 动态版（qcl: 10.4KB, qvm: 22.4KB） |
| **最大吞吐** | 批量100次提升 **29%**，单次提升 **34%** |
| **内存节省** | RSS降低 **61%**（1980KB→780KB），页错误降低 **42%** |
| **稳定性** | 5种极端场景全部通过（>1000指令、>64量子比特、>10000行、控制流、一致性） |
| **正确性** | 字节码输出与基线100%一致 |

### 优化收益摘要
- **I/O优化（mmap）：** 减少系统调用、零拷贝、内存↓61%
- **编译优化（-O2）：** 单次性能↑34%，批量↑29%
- **小buffer：** 栈内存↓60%，缓存友好
- **静态编译：** 零运行时依赖（代价：体积↑50倍，适合离线/容器隔离）
