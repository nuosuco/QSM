# scripts - 构建和工具脚本文档

## 概述

本目录包含QEntL项目的各种构建脚本、工具脚本和自动化脚本的文档。

## 脚本类型

- **构建脚本** - 项目编译和构建
- **安装脚本** - 系统安装和配置
- **测试脚本** - 自动化测试执行
- **工具脚本** - 开发辅助工具

## 主要脚本

- `build_all.bat` - 完整构建脚本
- `install.qentl` - 安装配置脚本
