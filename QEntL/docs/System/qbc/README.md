# qbc - 量子字节码文档

## 概述

QBC (Quantum Bytecode) 是QEntL的量子字节码格式，是编译器输出和虚拟机执行的中间代码格式。

## 核心特性

- **QENT魔数** - 文件格式标识 (51 45 4E 54)
- **高压缩比** - 约90%的压缩效率
- **量子指令集** - 支持量子计算操作
- **跨平台兼容** - 支持多种硬件架构

- QBC文件格式规范
- 字节码指令集定义
- 编译验证报告
