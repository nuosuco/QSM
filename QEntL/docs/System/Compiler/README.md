# QEntL Compiler 编译器文档

本目录包含QEntL编译器的完整文档。

## 文档列表

- `COMPILER_BUILD_ARCHITECTURE.md` - 编译器构建架构和实际构建记录
- `compiler_implementation_plan.md` - 编译器实现计划

## 编译器特性

- QEntL源码编译到QBC字节码
- 支持量子编程语法
- 优化的字节码生成
- 完整的错误处理

## 使用方法

```batch
# 编译QEntL源文件
qentl_Compiler.bat source_file.qentl
```
