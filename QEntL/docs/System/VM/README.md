# QEntL Virtual Machine 虚拟机文档

本目录包含QEntL虚拟机的完整文档。

## 文档列表

- `VM_BUILD_ARCHITECTURE.md` - 虚拟机构建架构和实际构建记录
- `vm_implementation_plan.md` - 虚拟机实现计划

## 虚拟机特性

- QBC字节码文件执行
- 量子状态管理
- 魔数验证和内容分析
- 完整的调试支持

## 使用方法

```batch
# 运行QBC字节码文件
qentl_vm.bat bytecode_file.qbc
```

## 核心功能

- QBC文件加载与验证
- 字节码解析与执行
- 量子指令处理
- 性能分析与监控
