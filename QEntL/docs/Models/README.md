# Models 四大核心模型文档

本目录包含QEntL四大核心模型的详细文档。

## 四大核心模型

### QSM - 量子叠加态模型
量子状态管理和叠加态处理的核心模型。

### WeQ - 量子通讯模型  
基于量子纠缠的通信和网络协议模型。

### SOM - 量子平权经济模型
量子计算资源的分配和经济管理模型。

### Ref - 量子自反省模型
系统自我优化和学习的反馈机制模型。

## 目录结构

- `QSM/` - 量子叠加态模型文档
- `WeQ/` - 量子通讯模型文档
- `SOM/` - 量子平权经济模型文档
- `Ref/` - 量子自反省模型文档

## 模型特性

- 量子态管理
- 纠缠网络构建
- 资源自适应分配
- 智能学习优化
