# QSM项目实践学习指南

## 📁 学习目标

深入学习和理解QSM项目的完整实现，包括所有架构、设计、代码细节，为实际参与项目开发奠定基础。

---

## 🏗️ QSM项目整体架构

### 核心组成
1. **QEntL** - 量子叠加态操作系统
2. **量子动态文件系统** - QDFS（Quantum Dynamic File System）
3. **量子神经网络系统** - 量子感知、推理、记忆、学习
4. **量子编译器和虚拟机** - QBC编译器 + QVM虚拟机
5. **量子GUI系统** - 量子桌面环境和界面系统

### 项目目录结构
```
/root/QSM/
├── core-foundation-documents/  # 核心基础文档（已学）
├── docs/                       # 开发文档（约110个）
├── QEntL/                      # 量子操作系统实现
├── QEntL_Lib/                  # 量子操作系统库
├── quantum_file_system/        # 量子动态文件系统
├── QSM_Quantum_Neural/         # 量子神经网络系统
├── QBC_Compiler/               # QBC编译器
├── QBC_VM/                     # QBC虚拟机
├── QSM_GUI/                    # 量子GUI系统
├── Models/                     # 各种模型定义
├── qim/                        # 量子图像处理模块
└── ...（其他组件）
```
