### 架构理解
```
.qentl源码 → QEntL编译器 → QBC量子字节码 → 三种部署
```

### 三种部署
1. **QEntL量子操作系统**（原生部署）
2. **QEntL量子虚拟机**（跨平台环境）

1. ✅ 过渡运行器（Python版）- 已完成
